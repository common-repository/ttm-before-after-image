Want to highlight the differences between two images? Makes it easy with TTM Before After plugin.

== Description ==

TTM Before After Image is plugin that allows you to easily display a comparison between two images by placing them side-by-side. It is very easy to use and does not require any technical knowledge from the end users. It supports both vertical and horizontal orientation. This is the best before and after image comparison plugin to show the differences between two images by placing them side by side.

[Try out the demo](https://themetechmount.com/ttm-before-after/ "Live Demo") to see it in action!

**Features**

* Horizontal / Vertical Orientation
* Handle movement control - Swipe or Hover
* Customizable before and after labels
* Typography and color options
* Responsive and functional on all devices.
* Easy and clean user interface.

== Installation ==

= Download & Upload: =
1. Download the **TTM Before After Image** plugin (*.zip file*)
2. In your Admin area, go to the Plugins menu and click on **Add New** button
3. Click on **Upload Plugin** button and choose the **`ttm-before-after-image.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **ttm-before-after-image** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress

Access the sliders by going to 'TTM Before After' menu.
